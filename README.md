# Моніторинг мережевих пристроїв

Веб-сервіс для автоматичного моніторингу доступності мережевих пристроїв у реальному часі.

## Стек технологій

| Рівень | Технологія |
|--------|-----------|
| Серверна частина | Python 3.11+, FastAPI, Uvicorn |
| Клієнтська частина | HTML5, CSS3, Vanilla JS, Chart.js |
| База даних | SQLite (вбудована, без зовнішніх залежностей) |
| Оновлення в реальному часі | WebSocket |
| Опитування | ICMP ping (subprocess) |

## Структура проєкту

```
hackathon/
├── backend/
│   ├── main.py          # FastAPI-сервер, REST API, WebSocket
│   ├── database.py      # Операції з SQLite
│   ├── poller.py        # Адаптивний алгоритм опитування
│   ├── models.py        # Pydantic-моделі
│   └── requirements.txt
├── frontend/
│   ├── index.html       # Інформаційна панель (mobile-first)
│   ├── css/style.css    # Адаптивні стилі
│   └── js/app.js        # WebSocket-клієнт, логіка UI
├── algorithm.md         # Опис алгоритму опитування
└── TASK_SPEC.md         # Технічне завдання
```

## Запуск

### 1. Встановлення залежностей

```bash
cd backend
pip install -r requirements.txt
```

### 2. Запуск сервера

```bash
cd backend
uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```

### 3. Відкрити у браузері

```
http://localhost:8000
```

## API

| Метод | Маршрут | Опис |
|-------|---------|------|
| GET | `/api/devices` | Список усіх пристроїв |
| POST | `/api/devices` | Додати пристрій |
| GET | `/api/devices/{id}` | Деталі пристрою |
| PATCH | `/api/devices/{id}` | Оновити пристрій |
| DELETE | `/api/devices/{id}` | Видалити пристрій |
| GET | `/api/devices/{id}/history` | Історія статусів (24 год) |
| GET | `/api/devices/{id}/uptime` | Відсоток доступності |
| WS | `/ws` | WebSocket-стрім оновлень |

## Алгоритм опитування

Використовується **адаптивне інтервальне опитування**. Детальний опис, порівняння з альтернативами та оцінку ефективності див. у файлі [algorithm.md](algorithm.md).

## Реалізовані функції

### Обов'язкові
- ✓ Ф1 — Ping-перевірка доступності (ICMP)
- ✓ Ф2 — Панель моніторингу з переліком пристроїв та статусами
- ✓ Ф3 — Сповіщення в інтерфейсі при зміні статусу
- ✓ Ф4 — Додавання, редагування, видалення пристроїв
- ✓ Ф5 — Опис алгоритму опитування з обґрунтуванням

### Додаткові
- ✓ Д1 — Кольорова індикація: зелений / червоний / жовтий / сірий
- ✓ Д2 — Графік RTT за 24 години для кожного пристрою
- ✓ Д4 — Пошук та фільтрація за типом і статусом
- ✓ Д5 — Відсоток доступності за 24 години
